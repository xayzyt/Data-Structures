# Data Structures

用 **C 语言** 实现数据结构与基础算法的练习仓库。它主要记录我对顺序存储、链式存储、树结构、图结构和表达式处理等基础内容的学习过程。

## 仓库定位

这个仓库不是为了展示复杂业务，而是为了证明三件基础能力：

- 能用 C 写清楚结构体、指针和动态存储关系。
- 能理解常见数据结构的存储方式和操作接口。
- 能把算法过程拆成可调试、可验证的小程序。

## 内容导览

| 主题 | 对应练习 |
| --- | --- |
| C 基础与指针 | 函数、数组、字符串、指针 |
| 线性结构 | 顺序表、单链表、双向链表、循环链表 |
| 栈与队列 | 栈、队列、循环队列、链式队列 |
| 树结构 | 二叉树、线索二叉树、哈夫曼树、树/森林与二叉树转换 |
| 图结构 | 图的基本概念与存储结构 |
| 算法应用 | 表达式求值、快慢指针等典型问题 |

## 推荐阅读方式

1. 从 `2.顺序表/` 和 `3.链表/` 看线性表基础。
2. 再看 `6.栈队列/`、`7.循环队列/`、`8.队列的链式结构/`。
3. 最后看 `10.树_1/` 到 `15.图的基本概念和存储结构/`。

每个目录基本对应一个独立 Visual Studio 小工程，核心逻辑通常在 `main.c` 或 `test.c` 中。

## 面试时我会重点讲

- 顺序存储和链式存储的取舍。
- 指针、结构体和内存分配在数据结构实现中的作用。
- 栈/队列在表达式处理和任务调度思维中的联系。
- 树和图为什么是后续嵌入式算法、路径规划和状态搜索的基础。
