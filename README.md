# Data Structure Practice

这个仓库用于记录数据结构学习和 C 语言实现练习，重点是理解常见数据结构的存储方式、操作接口和复杂度。

## 内容方向

- 线性表、链表、栈、队列
- 树、二叉树、图等基础结构
- 查找、排序和基础算法
- C 语言实现与调试记录

## 学习目标

- 掌握常见数据结构的核心思想
- 能够用 C 语言实现基础结构和操作
- 建立后续嵌入式开发、算法题和系统编程需要的基础
