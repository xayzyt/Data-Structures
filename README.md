# Data Structures

C 语言数据结构练习仓库。

![C](https://img.shields.io/badge/C-00599C?style=flat-square&logo=c&logoColor=white)
![Visual Studio](https://img.shields.io/badge/Visual_Studio-5C2D91?style=flat-square&logo=visualstudio&logoColor=white)

## 内容

| 目录 / 主题 | 内容 |
| --- | --- |
| `0.函数-数组-字符串/` | 函数、数组、字符串 |
| `1.pointer/` | 指针 |
| `2.顺序表/` | 顺序表 |
| `3.链表/` | 单链表 |
| `4.快慢指针应用/` | 链表应用 |
| `5.双向链表/` | 双向链表 |
| `6.栈队列/` | 栈和队列 |
| `7.循环队列/` | 循环队列 |
| `8.队列的链式结构/` | 链式队列 |
| `9.表达式求职/` | 表达式求值 |
| `10.树_1/` ~ `14.树_5.../` | 树、二叉树、线索二叉树、哈夫曼树 |
| `15.图的基本概念和存储结构/` | 图的存储结构 |

## 说明

- 每个目录基本对应一个 Visual Studio 小工程。
- 主要代码在 `main.c` 或 `test.c` 中。
- 仓库包含部分编译输出和 IDE 生成文件。
